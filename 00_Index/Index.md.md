# Proyecto Mburicao

## ¿Qué es este proyecto?
Este documento es el índice principal de la documentación del proyecto
**Proyecto Mburicao**.

El objetivo de este vault es preservar el conocimiento del proyecto:
estructura, datos, notebooks, scripts y decisiones técnicas,
para poder reutilizarlo y entenderlo en el futuro.

## Cómo está organizada esta documentación
La estructura de esta documentación replica la estructura del repositorio
en GitHub.  
Cada carpeta documenta exactamente la carpeta homónima del proyecto.
